Compiled with g++.exe(Rev6, Built by MSYS2 project) 13.1.0

To compile, open cmd, go to 487A1 directory, 3 options:
	1.	type make, run complexcalc
	2.	g++ -o complexcalc main.c complex.c, run complexcalc
	3.  run build.bat

If complexcalc has
 	- no redirect, will start in interactive mode.
	- only input redirect, will output results to terminal.
	- only output redirect, will output results to file, after quit.
	- both i/o redirect, input from file and output to file.
